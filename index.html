EcoGenAI

This is an AI-powered predictive maintenance system developed for the aerospace industry to detect early signs of equipment wear and anomalies. Built using Python, FastAPI, and Streamlit, the system integrates acoustic and visual sensor data with machine learning models to analyze component health in real-time. The project was a finalist in the 2025 SustAInable Aviation Challenge, recognized for its innovative approach of AI and LLM. EcoGenAI not only enhances equipment reliability but also reduces maintenance costs by enabling proactive decision-making.

Video Streaming using CDN

This project focused on building a scalable, secure, and efficient content delivery network for on-demand video playback. Designed with a distributed architecture, the system uses a round-robin load balancer to route requests across multiple replica servers, ensuring high availability and reduced latency. Videos are streamed asynchronously over HTTP/2 and secured with HTTPS using self-signed SSL certificates. The architecture includes an origin server that manages content replication and a controller that orchestrates video requests.

IoT Based Weather Monitoring System

This project is a real-time environmental data acquisition project built using a Raspberry Pi 3 and various sensors. It captures key atmospheric parameters such as temperature, humidity, air pressure, and air quality using DHT11, BMP180, MQ135, and MCP3008 sensors interfaced via I2C, SPI, and UART protocols. The system streams the collected data live to the ThingSpeak cloud platform, enabling remote monitoring and visualization.

Root Cause Analysis Using RAG based Architecture

This project leverages Retrieval-Augmented Generation (RAG) architecture to identify and explain the causes of system failures using natural language processing. A custom pipeline was built where relevant logs and prior incident reports are indexed using vector embeddings. At runtime, the system retrieves the most contextually similar documents related to a failure query and feeds them into a language model to generate coherent, human-readable explanations. This approach not only enhances accuracy by grounding responses in real data but also enables scalable and explainable diagnostics
